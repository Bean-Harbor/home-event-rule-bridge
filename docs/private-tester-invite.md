# Private Tester Invite

Use these as short messages when asking a friend to try the bridge. Keep the request small: one install attempt, one Discord smoke run, and honest feedback.

## English

```text
Hey, I am testing a small Home Assistant + Discord tool.

The idea is simple: describe a HA rule in Discord, review a readable draft, and decide whether to show YAML, edit, confirm, or cancel. It runs in dry-run mode by default, so it should not change your HA files.

If you already run Home Assistant and Docker Compose, could you try the private tester guide once?

Guide:
https://github.com/Bean-Harbor/home-event-rule-bridge/blob/main/docs/private-tester-guide.md

What I care about most:
- Did setup make sense?
- Did doctor show ready: yes?
- Did the bot understand the smoke messages?
- Did any reply feel confusing or too confident?

Please do not share tokens or full .env files. A screenshot of the bot reply or a GitHub issue is enough.
```
